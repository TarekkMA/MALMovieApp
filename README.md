# MALMovieApp
###### TheMovieDb Android Client
***
####  **Instalition**
1. Import this project to your Android Studio
2. Create ``Secrets.java`` inside the root pacakge ``com.tmaproject.malmovieapp``
3. Get your api key from [**TheMovieDb Account  Page**](https://www.themoviedb.org/account/) 
4. Declare ``THEMOVIEDB_API_KEY`` String variable containing your api key from 
5. Let the project sync and Voila ! A fully working Movie Client Application
6. Contribute to the project
